CallbackApi

- This type of API is used on our website

- You can have it on your website

DragonApi as a CallbackApi - Methods

checkSqlStatus and checkWebServerStatus

These methods are used in our discord bot and are unlikely to be used by you.

They return 2 http codes: 200 if the request is successful or 520 if the request is unsuccessful

documentationRedirect

Redirect you to the api documentation page

testApi

This is required to check if the api is working or not

Returns response code 202

ownReponseCode

Your own response code

Arguments:

- responseCode (text)

Example:

- http://dragonfire.7m.pl/api/callbackApi.php?method=ownReponseCode&responseCode=202 returns response code 202

DragonApi as a CallbackApi - Events

Error 400 Bad request

This error occurs if argument is missing or invalid
